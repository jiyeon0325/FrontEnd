## 코딩애플 강의
# HTML/CSS All-in-one 

### 이미지 가운데 정렬
display: block;   <--- 가로행을 모두 차지 </br>
margin-left: auto; </br>
margin-right: auto </br>
```
.pic {
  style="display:block;
  margin-left:auto; 
  margin-right:auto"
}
```
### 글자 가운데 정렬
text-align: center
```
.title {
  text-align: center;
}
```
### div, p, h 가운데 정렬
#### display: block 이 기본으로 있음
margin-left: auto; </br>
margin-right: auto </br>
```
.box {
  margin-left: auto;
  margin-right: auto:
}
```
